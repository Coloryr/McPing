# McPing 
一个MC服务器ping插件

注：启动前请安装net8

## 启动
下载压缩包，解压到一个地方  
Windows双击运行`McPing.exe`生成默认配置  
Linux  
先安装dotnet
然后输入
```
dotnet McPing.dll
```
修改`config.json`中的`Group`运行的群与`Url`机器人地址  
重启程序

**Linux不会用就不要用Linux**

## 机器人链接
默认使用nonebot2 QQ官方机器人链接，若需要使用onebot，需要在配置文件调整  
nonebot2请安装插件[mcping_nonebot2](https://github.com/Coloryr/mcping_nonebot2)
